1. Introduction

Tapix is an AI-powered keyboard and text transformation app for iOS. It allows you to select text and transform it using artificial intelligence — rewrite, translate, summarize, fix grammar, and more. This Privacy Policy explains what data we collect, how we handle it, and what rights you have regarding your information.

2. Who We Are

Tapix is developed by Dmzlabz Inc., based in Israel.

Contact: vittra.app@gmail.com

3. Data We Collect and Process

a. Text Data Processed by AI

When you use an AI action, the selected text (up to 4,000 characters) is sent to third-party AI providers for processing. This text is transmitted directly to the provider and is not stored on our servers — we have no servers.

The AI providers are:

  • Anthropic (Claude API) — processes your text and returns results. Per Anthropic's API policy, inputs and outputs are not used for model training and are not retained beyond 30 days for trust & safety purposes.
  • Google (Gemini API) — used as a fallback when the primary provider is unavailable. Per Google's API terms, data is processed but not retained for model training.

No user identity, device ID, or personal metadata is attached to AI requests — only the text you selected and a system prompt describing the requested action.

b. Action History (Local Only)

We store a brief record of your AI actions locally on your device:

  • Action type and label
  • First 80 characters of input and output (preview only)
  • Timestamp and character counts

This data is stored in a local SQLite database on your device and is never transmitted to any server. It exists solely for your reference.

c. Usage Analytics

We use Firebase Analytics to collect anonymized usage data:

  • Screen views and feature usage events
  • Subscription tier (free or premium) as a user property
  • Standard device information collected by Firebase SDK (device model, OS version, anonymous instance ID)

No user text content is ever included in analytics events. Analytics can be considered anonymized as there are no user accounts.

d. Purchase Data

Subscription purchases are handled entirely by Apple through StoreKit. We receive transaction verification tokens from Apple to confirm subscription status. We do not collect or store any payment information (credit cards, billing addresses, etc.).

e. App Settings

Your preferences (display currency, translation language, notification settings, etc.) are stored locally on your device.

4. Keyboard Extension & Full Access

Tapix includes a custom keyboard extension that requires "Full Access" to function. Full Access is needed solely to make network requests to AI providers for text transformation.

When using the keyboard:

  • The keyboard reads text near your cursor to provide context for AI actions
  • Selected text is sent to AI providers only when you explicitly trigger an action
  • The keyboard does not log keystrokes
  • The keyboard does not collect passwords or sensitive field content beyond what you explicitly select for transformation
  • A brief action history entry (80-character preview) is saved locally for your reference

Without Full Access enabled, the keyboard cannot access the network and AI features will not function.

5. How We Use Your Data

  • AI text processing: To perform the text transformation you requested
  • Action history: To show you a local log of your recent actions
  • Analytics: To understand which features are used and improve the app
  • Purchase verification: To unlock premium features for subscribers
  • Settings: To remember your preferences

6. Data Storage & Security

  • All user data is stored locally on your device using iOS-native storage (SQLite/SwiftData and UserDefaults)
  • There is no cloud sync — your data stays on your device
  • We do not operate any servers that store user data
  • All network communications use HTTPS encryption
  • iOS Data Protection applies to all stored data
  • API configuration is managed securely via Firebase Remote Config

7. Data Retention

  • Local data (action history, settings): Retained on your device until you delete it or uninstall the app
  • AI provider data: Anthropic retains API data for up to 30 days for trust & safety; Google processes data transiently per their API terms
  • Firebase Analytics: Retained per Google's default data retention settings (14 months)
  • Uninstalling Tapix: Removes all locally stored data from your device

8. Data Transfers

When you use AI features, your selected text is transmitted to servers operated by:

  • Anthropic (United States) — for Claude AI processing
  • Google (United States) — for Gemini AI processing (fallback)
  • Google/Firebase (United States) — for analytics and configuration

These transfers are necessary to provide AI-powered text transformation. All transfers use HTTPS encryption.

11. Children's Privacy

Tapix does not knowingly collect personal information from children under 13. The app does not contain age-restricted content, but AI-generated text transformations may occasionally produce unexpected results. Parental guidance is recommended for young users.

12. AI Content Disclaimer

Text transformations are generated by third-party AI models. We include safety guidelines in our AI prompts, but we cannot guarantee that all AI-generated content will be appropriate in every context. You are responsible for reviewing AI-generated text before using it.
